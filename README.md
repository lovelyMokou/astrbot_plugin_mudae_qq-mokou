# 二次元笑传之抽抽Bot

基于NapCat和AstrBot的二次元抽卡QQ机器人插件。简单复刻了discord里mudae的功能，基本上是群聊版的集卡游戏。卡池里有来自bangumi的1w+角色。

## 部署

NapCat和AstrBot的部署请参考 [https://docs.astrbot.app/deploy/platform/aiocqhttp/napcat.html](https://docs.astrbot.app/deploy/platform/aiocqhttp/napcat.html)

## 参数设置

请在“插件-插件配置”中浏览并编辑默认游戏参数。

## 基本指令

**查看帮助**
```
@机器人 菜单
@机器人 帮助
```

**抽卡**
```
@机器人 抽卡
@机器人 ck
```

**查看后宫**
```
@机器人 我的后宫
@机器人 我的后宫 <页码>
```

**结婚**
给抽卡结果消息贴表情即可收集该角色。默认冷却一小时。

**离婚**
```
@机器人 离婚 <角色ID>
```

**设置最爱**
```
@机器人 最爱 <角色ID>
```
被设置为最爱的角色在清理后宫和重开时不会被离婚。

**查询角色**
```
@机器人 查询 <角色ID>
@机器人 搜索 <角色名称>
```

**许愿系统** 略微增加爆率
```
@机器人 许愿 <角色ID>
@机器人 愿望单
@机器人 删除许愿 <角色ID>
```

**交换角色**
```
@机器人 交换 <我的角色ID> <对方角色ID>
```
对方给交换请求消息贴表情即可完成交换。

## 管理员指令

**系统设置**
```
@机器人 系统设置 抽卡冷却 <秒数>
@机器人 系统设置 抽卡次数 <次数>
@机器人 系统设置 后宫上限 <数量>
@机器人 系统设置 抽卡范围 <范围>
```
后宫上限同时也是愿望单上限。

**清理后宫**
```
@机器人 清理后宫 <QQ号>
```

**强制离婚**
```
@机器人 强制离婚 <角色ID>
```

## 群主/超管指令

**刷新冷却**
```
@机器人 刷新 <QQ号>
```
刷新球。会重置抽卡次数和结婚冷却。

**终极轮回**（差不多重开，清除所有婚姻数据）
```
@机器人 终极轮回 确认
```
